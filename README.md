# Cartoon Jump Game

A simple 2D side-scrolling jumping game built with **HTML5 Canvas** and **JavaScript**, featuring:
- Cartoon character
- Cartoon obstacles
- Scrolling background
- Score tracking
- Start/Restart buttons
- In-game instructions
